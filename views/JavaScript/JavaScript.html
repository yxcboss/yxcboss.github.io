                # js作用域

                # 概述

                ** 程序代码中所用的名字可用的代码范围 ** js中作用域分两种：

                • 全局作用域
                • 局部作用域（函数作用域）

                # 全局作用域

                ** 作用于所有代码执行环境 **

                # 局部作用域（函数作用域）

                ** 作用于函数内的代码环境，就是局部作用域 **

                • 调用函数时创建函数作用域，函数执行完后，函数作用域销毁
                • 每次调用函数就会创建一个函数作用域，他们之间是相互独立的
                • 在函数作用域中可以访问到全局作用域，反之不能
                • 在函数作用域操作一个变量时，他会先从自身作用域中寻找，如果有则直接使用，如果没有则向上一级作用域寻找，直到找到全局作用域，如果全局作用域中依然没有找到，则会报错ReferenceError
                • 在函数中若要使用全局变量，可以使用window对象

                # 变量的作用域

                # 全局变量

                ** 在全局声明的变量叫全局变量 **

                • 全局变量在代码的任何位置都可以使用
                • 在全局作用域中用var声明的变量是全局变量

                # 局部变量

                ** 在局部声明的变量是局部变量 **

                • 局部变量只能在该函数内部使用
                • 在函数内部用var声明的变量是局部变量
                • 函数的形参就是局部变量

                全局变量和局部变量的区别

                • 全局变量：在任何一个地方都可以访问的到，只在浏览器关闭后才能被销毁，因此比较占内存
                • 局部变量：只在函数内部使用，在代码被执行结束之后，就会被销毁，因此更节省内存空间

                # 作用域链

                采取就近原则的方式来查找变量最终的值

                访问一个变量时，会先从自身作用域中寻找变量对象，是否有这个变量，如果没有，那么会继续向上一级作用域中寻找，直到全局作用域。这个变量查找机制就叫作作用域链
