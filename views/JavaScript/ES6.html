                # 解构赋值

                • 解构赋值是对赋值运算符的扩展，针对数组或者对象进行模式匹配，然后对其变量赋值
                  let arr = [1,2,3];
  let [a,b,c] = arr;
  console.log(a,b,c) //1,2,3
  let obj = {
    name:'yxc',
    age:18
  }
  let {name,age} = obj;
  console.log(name,age);//yxc,18

                # Array.from()

                • 将类数组或可迭代对象转化成数组
                  let obj = {0:'a',1:'b',2:'c',length:3}
  console.log(Array.from(obj))
  //[a,b,c]

                注意哦

                • 属性名必须为数值或可转化成数值的
                • 没有length转化成空数组
